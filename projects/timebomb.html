Project Overview

TimeBomb is an innovative IoT time capsule prototype that allows users to lock items until a specific future date. This digital lock system combines hardware and software to create a secure time-based access control mechanism.

Status: 5 prototypes made

How It Works

The TimeBomb system uses a date-based locking mechanism. Users can set a specific unlock date, and the device will remain locked until that date arrives. The system features multiple modes for different use cases:

Lock Procedure: Set today's date, choose the unlock date, and engage the lock. The system validates the dates and begins the countdown.

Unlock Procedure: Once the unlock date is reached, the system automatically transitions to the unlocked state, allowing access to the contents.

Use Cases

This prototype explores the concept of delayed gratification and time-based access control. Perfect for:

  • Personal time capsules
  • Birthday surprises
  • Self-control mechanisms
  • Educational projects on IoT and time management
